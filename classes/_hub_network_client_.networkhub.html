Class NetworkHub

A network hub client. This type connects to a remote web socket running the smoke signalling protocol. For a pre-built web socket host, see the smoke-hub project.

example

$ npm install smoke-hub -g
$ smoke-hub --port 5000
example

const hub = new NetworkHub('ws://localhost:5000')
