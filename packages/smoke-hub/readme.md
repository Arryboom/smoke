# Smoke-Hub

The Smoke-Node Signalling hub. Provides WebRTC signalling, address allocation and message forwarding for [Smoke](https://github.com/sinclairzx81/smoke).

```
$ npm install smoke-hub -g
```
```
$ smoke-hub --port 5000
```
